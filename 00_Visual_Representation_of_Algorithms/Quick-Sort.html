Quick Sort

Divide & Conquer · Pivot-Based Sorting

How Quick Sort Works
1.

Choose a Pivot
Pick one element as the pivot (last element in this demo).

2.

Partition
Move elements smaller than pivot to left, larger to right.

3.

Place Pivot
Pivot lands at its final sorted position after partitioning.

4.

Recurse
Repeat on left and right sub-arrays until everything is sorted.

Pseudocode
function quickSort(arr, low, high): if low < high: // base case pi = partition(arr, low, high) // pivot index quickSort(arr, low, pi-1) // left sub-array quickSort(arr, pi+1, high) // right sub-array   function partition(arr, low, high): pivot = arr[high] // last element as pivot i = low - 1 // boundary pointer for j = low to high-1: if arr[j] <= pivot: // goes to left zone? i++ swap(arr[i], arr[j]) // move to left zone swap(arr[i+1], arr[high]) // place pivot correctly return i+1 // pivot's final position
